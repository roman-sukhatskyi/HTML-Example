Универсальный селектор

        Кроме селекторов тегов, классов и идентификаторов в css также есть так называемый
        универсальный селектор, который представляет знак звездочки (*). Он применяет стили
        ко всем элементам на html-странице:
        *{
            background-color: red;
         }
