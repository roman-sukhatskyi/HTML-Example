Селекторы элементов одного уровня

        Селекторы элементов одного уровня или смежных элементов позволяют выбрать элементы,
        которые находятся на одном уровне вложенности.Иногда такие элементы еще называют
        сиблинги (siblings) или сестринскими элементами.

        Чтобы стилизовать первый смежный элемент по отношению к определенному элементу,
        используется знак плюса +:
