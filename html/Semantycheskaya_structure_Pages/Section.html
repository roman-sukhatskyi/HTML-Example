Элемент section

Элемент section объединяет связанные между собой куски информации
        html-документа, выполняя их группировку. Например, section может включать
        набор вкладок на странице, новости, объединенные по категории и т.д.

        Каждый элемент section должен быть идентифицирован с помощью заголовка h1-h6.
