С помощью элемента datalist мы можем задать набор цветов,
        из который пользователь может выбрать нужный:
    
        Каждый элемент option в datalist должен в качестве значения принимать
        шестнадцатеричный код цвета, например, "#0000FF".
        После выбора цвета данный числовой код устанавливается в качестве значения
        в элементе input.

Поля для ввода url, email, телефона
