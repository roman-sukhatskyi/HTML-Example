Ползунок

        Ползунок представляет шкалу, на которой мы можем выбрать одно из значений.
        Для создания ползунка применяется элемент input с атрибутом type="range".
        Во многом ползунок похожна простое поле для ввода чисел. Он также имеет
        атрибуты min, max, step и value
